Usage Guidelines
Do
Use 1.8px stroke for all UI icons
Use Light weight for nav/toolbar
Use Duotone for dashboard features
Size: 16px nav, 20px mobile, 24px feature
Color: text-3 default, accent active
Don't
Don't use Lucide, Heroicons, or Feather
Don't mix icon libraries in the same view
Don't use filled icons for navigation
Don't exceed 2px stroke weight
Don't use full opacity — keep icons at 0.5–0.8
Recommendation: Start with @phosphor-icons/react for general UI. Supplement with Hugeicons' free legal category for domain-specific icons (legal documents, court, hearings). Use Phosphor's Duotone style for dashboard stat card icons and empty state illustrations.
